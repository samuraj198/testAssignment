# Тестовое задание для компании AmoPoint

### Как запустить laravel проект:
- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate (В качестве локального окружения решил использовать OpenServer для MySQL сервера)

## Задание 1:
- Консольная команда - **app:getAPIData**. Берет цену биткойна и записывает в базу
- Чтобы получить json с данными из этой базы нужно перейти на маршрут /crypto **(127.0.0.1:8000/crypto)**
## Задание 2:
- Скрипт, который нужно прописывать в консоли сайта (F12->console) находится в js файле по пути **public/js/test2.js**
## Задание 3:
- Скрипт, который нужно прописывать в консоли сайта (F12->console) находится в js файле по пути **public/js/test3.js**
- Этот скрипт берет данные с сайта и передает их бэку(laravel)
- Бэк(laravel) берет данные от js и сохраняет их в базу в таблицу visits
- Графики с посещениями выведены по маршруту /dashboard **(127.0.0.1:8000/dashboard)**. 
  Но этот маршрут доступен только авторизованным пользователям
- **127.0.0.1:8000/register** - регистрация
- **127.0.0.1:8000/login** - авторизация
- Графики построены с помощью библиотеки **Chart.js**
